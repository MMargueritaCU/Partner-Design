DISPLAY

PROPERTY

The display property specifies if/how an element is laid out on the page.

It specifies the display behavior (the type of rendering box) of an element.

In HTML, the default display property value is taken from the HTML specifications or from the browser/user default style sheet.

DISPLAY: IN-LINE;

Elements that are placed side by side, are inline elements ( use the <span> tag ).

Take space only as much as they need and do not accept width or height properties, and top-bottom margin

Can be a parent of other inline elements

  • IN-LINE EX.

DISPLAY: BLOCK;

The element is displayed as a block, as paragraphs and headers have always been.

Elements that each starting with a new line are called block-level elements

A block has some whitespace above and below it and tolerates no HTML elements next to it, except when ordered otherwise (by adding a float declaration to another element, for instance).

  • BLOCK EX.

DISPLAY: IN-LINE BLOCK;

The element shows both the characteristics of inline and block-level elements.

In other words, we can think of an inline element, that width & height properties can be set, or we can think of a block-level element, that doesn’t have to start with a new line.

Now they can be placed side by side and also width & height properties can be set.

  • IN-LINE

    BLOCK EX.

DISPLAY: NONE;

Hiding an element can be done by setting the display property to none. The element will be hidden, and the page will be displayed as if the element is not there.

This is commonly used with JavaScript to hide and show elements without deleting and recreating them.

The <script> element uses display: none; as default.

FLEX

The flex CSS property sets how a flex item will grow or shrink to fit the space available in its flex container. It is a shorthand for flex-grow, flex-shrink, and flex-basis.

For most purposes, authors should set flex to one of the following values: auto, initial, none, or a positive unitless number.

The flex property may be specified using one, two, or three values.
